# Quick Start Guide

Get running in **30 seconds**!

## Just Run It

**Windows:** Double-click `run.bat`

**Mac/Linux:** Run `./run.sh`

That's it! The smart launcher will:
- Show you what's installed and what's missing
- Offer to install dependencies automatically
- Guide you through any needed setup
- Start the metadata extractor when ready

---

## Or Use Python Directly

```bash
python start.py
```
